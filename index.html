Política de Privacidad – Sport Like Skill

Última actualización: julio 2026

1. Información que recopilamos

La Skill de Sport Like NO recopila ni almacena datos personales, de cuenta o de identificación de los usuarios. La skill funciona como un catálogo de consulta por voz para una tienda de artículos deportivos.

Durante el uso de la skill se procesa únicamente lo siguiente, de forma temporal y no persistente:

  • La frase de voz que el usuario dice a Alexa (procesada por Amazon Alexa según su propia política de privacidad)
  • Consultas de productos, categorías, sucursales, stock y promociones enviadas a nuestra API pública de catálogo
  • Para preguntas abiertas del usuario, el texto de la consulta puede enviarse a un servicio de inteligencia artificial de terceros (DeepSeek) únicamente para generar una respuesta conversacional; este texto no incluye datos personales del usuario y no se almacena por parte de Sport Like

2. Información que NO recopilamos

  • Nombre, correo electrónico o datos de contacto
  • Ubicación geográfica del dispositivo
  • Datos de pago o facturación
  • Historial de conversaciones entre sesiones

3. Uso de la información

Las consultas de voz se utilizan exclusivamente para responder en tiempo real sobre el catálogo de productos, categorías, existencias, sucursales y promociones de Sport Like. No se guarda ningún registro asociado a un usuario específico entre una sesión y otra.

4. Servicios de terceros

La skill consulta una API propia de Sport Like para obtener información del catálogo e inventario, y de forma opcional un servicio externo de procesamiento de lenguaje natural (DeepSeek) para responder preguntas libres. Ninguno de estos servicios recibe datos de identificación del usuario de Alexa.

5. Datos de menores

Esta skill no está dirigida a niños menores de 13 años y no recopila intencionalmente información de menores.

6. Cambios a esta política

Esta política puede actualizarse periódicamente. Cualquier cambio se reflejará en esta misma página con la fecha de actualización correspondiente.

7. Contacto

Para dudas sobre esta política de privacidad, puedes contactar al desarrollador a través del repositorio de GitHub de este proyecto.
